# Manzel Technologies Platform

## Overview
Manzel Technologies website showcasing digital twin solutions for real estate visualization and city simulation games. Built with React, Three.js, and Vite featuring interactive 3D elements and immersive experiences.

## Project Structure
- `src/` - Main source code
  - `components/` - React components including 3D canvas elements
  - `assets/` - Images, videos, and other static assets
  - `constants/` - Configuration data
  - `hoc/` - Higher-order components
  - `utils/` - Utility functions
- `public/` - Public static files (robots.txt, sitemap.xml)
- `dist/` - Build output

## Tech Stack
- React 18
- Vite (development server and build tool)
- Three.js with @react-three/fiber and @react-three/drei
- Framer Motion (animations)
- Tailwind CSS (styling)
- Material UI
- EmailJS (contact form)

## SEO Implementation
- Comprehensive meta tags for search engines
- Open Graph tags for social media sharing
- Twitter Card meta tags
- Structured data (JSON-LD) for organization schema
- robots.txt and sitemap.xml
- Canonical URL configuration

## Development
Run with: `npm run dev`
Build with: `npm run build`

## Configuration
- Vite configured on port 5000 with all hosts allowed for Replit proxy
- Uses legacy peer deps for npm install due to react-tilt compatibility

## Design System
- **Primary Colors**: Cyan (#00f0ff), Purple (#8b5cf6), Orange (#ff6b35)
- **Background**: Deep navy/black gradients (#0a0a1a, #050816)
- **Style**: Dark tech aesthetic with glassmorphism, gradient accents, and professional typography
- **Inspiration**: bluedesignstudio.com - bold, vibrant, corporate professional
